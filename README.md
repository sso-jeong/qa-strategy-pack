# QA Strategy Pack (익명화 템플릿)

이 레포는 아래 목적의 실전 템플릿/샘플을 제공합니다.
- **리스크 기반 테스트 전략 수립**
- **테스트 케이스 구조/커버리지 설계**
- **릴리즈 Go/No-Go 판단(품질 게이트) 리포트**

## 구성
- `/templates` : 재사용 가능한 템플릿 모음
- `/examples` : 예시로 채워둔 샘플 문서

## 빠른 사용법(Quick Start)
1. `/templates`에서 필요한 템플릿을 복사합니다.
2. 서비스 요구사항/변경점을 기준으로 내용을 채웁니다.
3. `/examples`를 참고해 작성 수준/형식을 맞춥니다.
